# AI Supplement Files

| File or Folder | Role |
| --- | --- |
| `AI_USAGE_LOG.md` | Log of AI tools, prompts, outputs adopted, and affected files |
| `DATA_INVENTORY.md` | Data inventory copied from `project/data/README.md` |
| `code/` | Scripts used for SEC data collection, operating metric curation, DCF modeling, sensitivity analysis, dashboard data generation, workbook formatting, and report export |

This folder is intended to support the course requirement to disclose AI usage and submit the code used in place of, or alongside, Excel modeling.

Human group members remain responsible for final assumptions, interpretation, report wording, and presentation delivery.
